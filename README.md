Repository ini berisi kumpulan *source code* (Google Apps Script) yang dikembangkan selama masa Praktek Kerja Lapangan (PKL) di Balai Besar Wilayah Sungai (BBWS) Mesuji Sekampung oleh mahasiswa Program Studi Teknik Informatika, Institut Teknologi Sumatera (ITERA).

Pengembangan *backend* sistem otomatisasi administrasi berbasis Google Workspace ini dibagi menjadi 3 topik pengerjaan utama untuk masing-masing mahasiswa.
